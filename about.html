Purpose

  • Purpose:The goal of this website is to help users better navigate the world of cryptocurrency. Whether you're buying or selling, this platform is designed to make those decisions easier by giving you accessible tools and insights. Beyond just transactions, the site aims to empower individuals with knowledge about the crypto market so they can make informed choices.

APIs Used

  • CoinGecko API: Provides live market data such as cryptocurrency prices, trends, and other market statistics.
  • GNews API: Supplies current news headlines from reputable media sources to give users broad financial context.

Technology Stack

  • FastAPI: Backend Python framework for creating high-performance APIs to serve market data, user preferences, and news.
  • httpx: Async HTTP client used for posting user preferences to the Supabase database.
  • requests: Used to make synchronous HTTP calls to APIs like CoinGecko and GNews for data retrieval.
  • CORS Middleware: Enables the frontend to securely interact with the backend server across different origins.
  • Supabase: Cloud database used to store and retrieve user preferences and data.
  • Visual Studio Code: Primary IDE used to develop and test the application.
  • Personal Laptop: All development work was performed locally before deployment.
